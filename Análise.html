Análise do Projeto de Teste

A empresa XPTO nos contratou para realizar a análise dos dados de viagens de Taxi por Aplicativo, dos anos 2009 a 2012.

Foram utilizadas as seguintes tecnologias na análise dos dados:

SQL Server e Power BI (para análise e normalização de dados)

Python (Google Colab para aplicar técnicas de Data Science em manipulação de massas de dados)

Git (Publicação do resultado no Github)
